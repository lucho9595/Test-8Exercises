## Prueba técnica - Ejercicios resueltos

En esta prueba técnica, se presentan 8 ejercicios de diversos tipos, como lógica, razonamiento, 
creación de páginas y manejo de bases de datos. 
Se ha optado por utilizar el lenguaje JavaScript (JS) y, en caso necesario, 
se han empleado frameworks relacionados con este lenguaje.
